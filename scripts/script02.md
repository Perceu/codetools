Date: 2023-12-06 13:39:59.928309
Category: scripts
Summary: Comandos para ler arquivos texto


para ler o final do arquivo
```shell
tail ./arquivos.txt
```

para ler do inicio do arquivo
```shell
head ./arquivos.txt
```

para arquivo inteiro
```shell
cat ./arquivos.txt
```
